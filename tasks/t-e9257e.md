## Description

> Imported from gh#108 — https://github.com/awizemann/scarf/issues/108

**Describe the bug**
The chat window produces a “chat connection failed” error if I switch away from the app and come back to it. 

**To Reproduce**
Steps to reproduce the behavior:
1. Start a chat
2. Write a message 
3. Press the send button. 
4. Switch to another app 
5. Switch back 
6. Observe error. 


**Expected behavior**
App should not error if I swap apps during a chat.
